02 — Descrição

Sobre o Architect

O Architect é um bot para Discord desenvolvido por Alzhayds, que oferece as seguintes funcionalidades:

  • Criação automatizada de servidores Discord utilizando Inteligência Artificial
  • Backup e restauração completa de estruturas de servidores
  • Sistema de proteção anti-nuke em tempo real
  • Templates prontos para diferentes tipos de comunidade
  • Gerenciamento de canais e cargos individualmente

Uso Aceitável

  • Compartilhar o bot com outros usuários de forma orgânica

05 — Privacidade

Dados Coletados

O Architect coleta e armazena os seguintes dados estritamente necessários para seu funcionamento:

  • ID do servidor (Guild ID) — para identificar e salvar backups
  • Estrutura do servidor — nomes de canais, cargos e categorias para o sistema de backup
  • Mensagens salvas — últimas 50 mensagens por canal, apenas quando o /backup é executado

Não coletamos senhas, tokens, dados pessoais dos membros ou informações de pagamento. Os dados são armazenados localmente e utilizados exclusivamente para as funcionalidades do bot.
