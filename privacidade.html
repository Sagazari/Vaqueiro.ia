Política de Privacidade

Última atualização: Março de 2025

O Architect coleta apenas os dados estritamente necessários para seu funcionamento. Nunca vendemos, compartilhamos ou usamos seus dados para fins publicitários.

1. Quem somos

O Architect é um bot para Discord desenvolvido e mantido por Alzhadys. Esta política descreve como coletamos, usamos e protegemos os dados dos servidores e usuários que utilizam o bot.

2. Dados que coletamos

O Architect coleta e armazena os seguintes dados:

DadoFinalidadeRetenção
ID do servidorVincular backups e configuraçõesAté remoção do bot
Nome do servidorIdentificação no painel de backupsAté remoção do bot
Estrutura do servidorBackup de cargos, canais e permissõesAté solicitação de exclusão
Preferência de idiomaPersonalização do bot por servidorAté remoção do bot
IDs de usuáriosAssociar cargos no backup (quando ativo)Junto ao backup do servidor

3. Dados que NÃO coletamos

  • Conteúdo de mensagens enviadas nos servidores
  • Dados pessoais como e-mail, telefone ou endereço
  • Histórico de atividades de membros
  • Dados de pagamento
  • Informações de perfil do Discord (avatar, bio, etc.)

4. Como usamos os dados

Os dados coletados são usados exclusivamente para:

  • Executar as funcionalidades do bot (backup, restauração, proteção)
  • Armazenar preferências de configuração por servidor
  • Melhorar a estabilidade e desempenho do serviço

Não utilizamos seus dados para publicidade, rastreamento ou qualquer outro fim comercial.

5. Armazenamento e segurança

Os dados são armazenados no MongoDB Atlas, um serviço de banco de dados em nuvem com criptografia em repouso e em trânsito. O acesso é restrito exclusivamente ao sistema do Architect.

6. Seus direitos

Você tem direito a:

  • Acessar os dados armazenados do seu servidor
  • Excluir todos os dados — basta remover o Architect do servidor
  • Solicitar exclusão manual via servidor de suporte

7. Serviços de terceiros

O Architect utiliza os seguintes serviços de terceiros:

  • Discord API — plataforma base do bot
  • Mistral AI — geração de estruturas com IA
  • MongoDB Atlas — armazenamento de dados

Cada serviço possui sua própria política de privacidade, que recomendamos consultar.

8. Contato

Para dúvidas, solicitações de exclusão ou qualquer questão relacionada à privacidade, entre em contato pelo nosso servidor de suporte.
