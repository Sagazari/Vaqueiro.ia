Política de Privacidade

Última atualização: Abril de 2025

O Architect coleta apenas os dados estritamente necessários. Nunca vendemos, compartilhamos ou usamos seus dados para fins publicitários.

1. Quem somos

O Architect é um bot para Discord desenvolvido e mantido por Alzhadys.

2. Dados que coletamos

DadoFinalidadeRetenção
ID do servidorVincular backupsAté remoção do bot
Nome do servidorIdentificaçãoAté remoção do bot
Estrutura do servidorBackupAté solicitação de exclusão
Preferência de idiomaPersonalizaçãoAté remoção do bot

3. Dados que NÃO coletamos

  • Conteúdo de mensagens
  • Dados pessoais como e-mail ou telefone
  • Histórico de atividades de membros
  • Dados de pagamento

4. Como usamos os dados

Os dados são usados exclusivamente para executar as funcionalidades do bot e melhorar a estabilidade do serviço.

5. Armazenamento

Os dados são armazenados no MongoDB Atlas com criptografia em repouso e em trânsito.

6. Seus direitos

  • Acessar os dados do seu servidor
  • Excluir todos os dados removendo o bot
  • Solicitar exclusão manual via suporte

7. Contato

Dúvidas sobre privacidade? Entre em contato.
